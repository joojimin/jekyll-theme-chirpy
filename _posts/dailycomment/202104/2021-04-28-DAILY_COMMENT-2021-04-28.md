### <span style="color: #aa88aa;">squash merge</span>
*   commit a+b+c를 합쳐서 새로운 m을 생성하여 target 브랜치에 추가(merge)
*   commit a+b+c를 합쳐서 새로운 commit abc를 생성하고 target 브랜치에 추가<br>
    ( squash and Merge )
*   feature 브랜치의 commit history를 합쳐서 깔끔하게 만들기위해 사용
